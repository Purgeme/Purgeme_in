Paint is bloated#

The MS Paint application is bloated and so I needed something better, something simpler, something less “bloated” and so I set out on a quest to make my own “unbloated” paint application from scratch.

Just kidding it was a college project that I completed with my team.

Basically it is a paint application written in c++ and uses OpenGL.

Features#

  1. You can draw ( I thought this is pretty important ).
  2. The ability to choose colors to your own preference.
  3. Different shaped brushes.
  4. The ability to change the brush size, as well as to make it hollow if needed.
  5. The ability to save the image ( still needs work though ).
